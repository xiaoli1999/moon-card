### 项目描述
- 定制中秋贺卡，预置祝福、团圆、思念三个贺卡主题；
- 用户只需上传头像，修改文案，简单调整位置即可快速定制出中秋贺卡。
- 支持预览、保存。
- 支持生成海报，分享给朋友。
- 支持中秋贺卡集功能，用户可观看他人定制的贺卡。
